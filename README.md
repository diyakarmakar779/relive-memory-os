# ReLive Memory OS

ReLive is an AI-powered voice memory app that transforms spoken moments into emotional stories with voice narration.

## What it does

- Capture voice memories using speech input
- Convert speech into emotional AI-generated stories
- Detect emotion (Love, Calm, Nostalgia, etc.)
- Generate voice narration using ElevenLabs
- Display memories in a timeline UI

## Current Features

- Voice-to-text memory creation
- AI story generation
- Audio playback for each memory
- Timeline-based memory feed
- Basic emotion tagging (from AI response)
